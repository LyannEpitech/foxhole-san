# Foxhole Swiss Army Knife

Outil de planification pour Foxhole :

- **Routes logistiques** — planifier et optimiser les trajets de ravitaillement.
- **Attaques** — préparer des opérations offensives.
- **Chaînes de production** — planifier la construction de chaînes de production en tenant compte des prérequis, des besoins des bâtiments et des ressources.

> Projet en cours d'initialisation.
